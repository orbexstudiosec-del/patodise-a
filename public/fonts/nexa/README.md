# Nexa fonts

Coloca aquí los archivos **woff2** de Nexa con estos nombres exactos:

- `Nexa-Light.woff2`    (peso 300)
- `Nexa-Regular.woff2`  (peso 400)
- `Nexa-Bold.woff2`     (peso 700)
- `Nexa-Heavy.woff2`    (peso 900)

Nexa es una fuente comercial de Fontfabric. Opciones:

1. **Nexa Free** — Fontfabric distribuye una versión gratuita con Light + Bold en
   https://www.fontfabric.com/fonts/nexa/  (haz scroll hasta "Get Nexa Free")
   Conviértelos a `.woff2` con https://transfonter.org si vienen en .otf/.ttf.

2. **Nexa completo** — requiere licencia comercial en
   https://www.fontfabric.com/fonts/nexa/

Mientras no haya archivos aquí, el sitio cae al fallback Inter / system-ui.
